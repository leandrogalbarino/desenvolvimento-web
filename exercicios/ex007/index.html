QUEM SOU EU?

Neste site, não apenas compartilharei soluções para as dúvidas que tive, mas também pedirei ajuda quando necessário! Se está em busca de um professor/aluno que, assim como você, está ansioso para discutir programação, este é o lugar certo para você!
